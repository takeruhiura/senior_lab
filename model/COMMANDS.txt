________________________________________________________________________________
PARAMETERS EXPLAINED
________________________________________________________________________________

--mode          Operation mode
                ‚Ä¢ image     = Single image detection
                ‚Ä¢ video     = Process video file
                ‚Ä¢ webcam    = Real-time webcam
                ‚Ä¢ export    = Export for FPGA
                ‚Ä¢ benchmark = Test speed

--source        Input source
                ‚Ä¢ Image path: "myimage.jpg"
                ‚Ä¢ Video path: "myvideo.mp4"
                ‚Ä¢ Webcam: 0 (or 1, 2 for other cameras)

--img-size      Input image size (affects speed)
                ‚Ä¢ 320  = Fastest (75 FPS) ‚≠ê RECOMMENDED FOR FPGA
                ‚Ä¢ 416  = Balanced (45 FPS)
                ‚Ä¢ 640  = Most accurate (25 FPS)

--conf          Confidence threshold
                ‚Ä¢ 0.15 = More detections (may have false positives)
                ‚Ä¢ 0.25 = Default (balanced)
                ‚Ä¢ 0.35 = Fewer detections (high confidence only)

--save-video    Save output video (flag, no value needed)

--export-format Format for FPGA export
                ‚Ä¢ onnx     = Universal (recommended)
                ‚Ä¢ openvino = Intel FPGAs
                ‚Ä¢ engine   = TensorRT (NVIDIA)

________________________________________________________________________________
TROUBLESHOOTING
________________________________________________________________________________

Problem: No detections
Solution: Lower confidence threshold
   python3 detect.py --mode image --source IMAGE.jpg --conf 0.15

Problem: Too slow
Solution: Use smaller image size
   python3 detect.py --mode image --source IMAGE.jpg --img-size 320

Problem: Can't find model
Solution: Make sure best.pt is in the same folder as detect.py

Problem: ImportError
Solution: Install dependencies
   pip install -r requirements.txt
